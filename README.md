# God Sandbox Framework (GSF)

![God Sandbox Framework](https://vampire-chan.github.io/GodSandboxFramework/hero.png)

Welcome to the official documentation and landing page repository for the **God Sandbox Framework**.

The God Sandbox Framework is a premium, data-driven AAA open-world engine template built on Unreal Engine 5.7. It completely replaces hardcoded blueprints, traditional State Machines, and spaghetti C++ with a robust XML-driven architecture, a native Virtual File System (VFS), a custom ScriptVM, and lightning-fast C++ Slate UI components.

**Build Worlds. Not Boilerplate.**

## 🚀 About This Repository

This repository contains the React/Vite source code for the God Sandbox Framework's public-facing website and documentation hub. 

- **Tech Stack:** React 18, Vite, React Router, Vanilla CSS
- **Deployment:** Automatically deployed to GitHub Pages via GitHub Actions upon pushing to the `main` branch.

### Local Development

If you wish to run the documentation website locally:

```bash
# Install dependencies
npm install

# Run the local Vite dev server
npm run dev
```

### Note on Engine Source Code
This repository **only** contains the web documentation and landing page infrastructure. The core C++ Unreal Engine 5.7 monolith for the God Sandbox Framework is a separate licensed commercial product.
